# Troubleshooting Guides

## Cannot Join Video Room
- Ensure network allows `meet.jit.si`.
- Reload the page to reinitialize the Jitsi API.

## Document Fails to Save
- Check that `slug` is unique.
- Verify you are authenticated; check session.

## Enrollment Issues
- Confirm module exists and is active.
- Retry enrollment; contact admin if status remains unchanged.

## WhatsApp/Discord Mirror Not Working
- Validate webhook URLs and tokens.
- Confirm channel mirroring flags are enabled.
